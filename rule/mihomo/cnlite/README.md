# ChinaLite

- 这个规则用于收集一些直连的漏网之鱼规则，因为有些个人博客或小众的论坛并没有被大规则收录，规则如果设置的是黑名单可能没有问题，但如果是白名单就会出现问题，因为国内直连访问比代理更快就会出现连接很慢的问题。
