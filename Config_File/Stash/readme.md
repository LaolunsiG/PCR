# XiaoE_Stash

IOS 巨魔用户可能无法使用覆写，正常的分流可以使用，推荐入手正版30￥，自行配置订阅链接。Stash 的 Wiki 好奇怪，感觉少了很多东西，无法合理的进行配置。完全兼容 Clash Premium 的配置

- 主要配置了地区优选，故障转移，负载均衡，并为每个策略组加上了图标，图标来源**[icon](https://github.com/LaolunsiG/XiaoE_PCR/tree/main/icons)**
